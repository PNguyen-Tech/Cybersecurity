# GRC & Cloud Security Study Notes

This directory serves as a centralized knowledge base for Governance, Risk, and Compliance (GRC) concepts, cloud security architectures, and academic coursework. 

The materials documented here run in parallel with the Master of Science in Management Information Systems (MIS) program at the University of Houston's C.T. Bauer College of Business. 

## 📚 Directory Contents

### Frameworks & Auditing Standards
* **NIST SP 800-53:** Security and privacy controls for federal information systems.
* **SOC 2 (System and Organization Controls):** Trust Services Criteria focusing on Security, Availability, and Confidentiality.
* **ISO/IEC 27001:** Information security management systems (ISMS).

### Cloud Security & Engineering
* **Microsoft Azure Administration:** Configurations, identity access management, and security controls for cloud-native environments.
* **Data & Automation Security:** Secure practices for SQL database management and Python scripting within compliance boundaries.

### Certification Preparation
* **CISA (Certified Information Systems Auditor):** Study materials, control mapping, and domain breakdowns in preparation for Fall 2026.
* **Microsoft Certifications:** Technical exam prep notes bridging the gap between infrastructure and compliance.
